# What each specialist wants to hear first

Specialists are pattern-matchers in a narrow lane. Leading with what *they* care about saves minutes. One or two lines each — lead the Visit Sheet with these when the specialty is known.

| Specialist | Leads with | Worth bringing |
| --- | --- | --- |
| **Cardiology** (heart) | What the chest symptom feels like and what brings it on | Blood-pressure history, what you were doing when it hit, how much exercise you can do |
| **Neurology** (brain / nerves) | The timeline and pattern of episodes | When it happens, how long it lasts, triggers, a symptom diary if you kept one |
| **Dermatology** (skin) | How a spot or rash has *changed* | Photos over time, when it appeared, itch / pain / bleeding, sun history |
| **Gastroenterology** (gut) | The pattern around eating and bowel habits | Trigger foods, timing vs. meals, changes in stool, weight changes |
| **Rheumatology** (joints / autoimmune) | Which joints, and the morning-stiffness pattern | Which joints, how long stiffness lasts in the morning, swelling, fatigue, family history |
| **Endocrinology** (hormones) | The symptom cluster and any relevant lab results | Weight, energy, temperature tolerance, recent bloodwork |
| **Orthopedics** (bones / joints) | The injury or mechanical story | How it happened, which movements hurt, what you can't do now, prior imaging |
| **ENT** (ear / nose / throat) | Which side, how long, and the trigger | Duration, one side vs. both, effect on hearing / breathing / swallowing |
| **Pulmonology** (lungs) | The breathing pattern and triggers | When breathlessness hits, cough details, smoking / exposure history |
| **Urology / Gynecology** | The pattern and timing of the symptom | Cycle timing where relevant, urinary pattern, what's changed |

If the specialty isn't known yet, default to a clean general timeline (see [`opqrst-symptom-framing.md`](opqrst-symptom-framing.md)) plus the person's top question.
